Linux 中的 `date` 命令不仅可以显示当前的系统日期和时间，还可以进行日期格式化、计算过去/未来的日期，甚至修改系统时间。它是脚本编写和系统管理中非常实用的工具。

以下是 `date` 命令的详细用法指南：

### 1. 基础用法
直接输入 `date` 会以默认格式显示当前系统时间：
```bash
date
# 输出示例: Tue Jan 13 10:54:32 CST 2026
```

### 2. 自定义显示格式 (最常用)
使用 `+` 号后跟格式说明符，可以自定义输出格式。格式说明符必须以 `%` 开头。

**常用的格式说明符：**
*   **日期类：**
    *   `%Y`：四位年份 (如 2026)
    *   `%m`：月份 (01..12)
    *   `%d`：一个月中的第几天 (01..31)
    *   `%F`：完整的日期 (相当于 `%Y-%m-%d`)
    *   `%A`：星期几的全称 (如 Tuesday)
    *   `%a`：星期几的简称 (如 Tue)
*   **时间类：**
    *   `%H`：小时 (00..23)
    *   `%M`：分钟 (00..59)
    *   `%S`：秒 (00..60)
    *   `%T`：时间 (相当于 `%H:%M:%S`)
*   **其他：**
    *   `%s`：自 1970-01-01 以来的秒数 (Unix 时间戳)

**示例：**
```bash
# 输出 YYYY-MM-DD HH:MM:SS 格式
date +"%Y-%m-%d %H:%M:%S"
# 2026-01-13 10:55:01

# 只输出当前年份和周数
date +"Year: %Y, Week: %V"
```

### 3. 显示特定时间或计算日期 (使用 `-d` 选项)
`-d` (或 `--date`) 选项功能极其强大，它允许你使用“人类可读的字符串”来查询时间，而不需要改变当前系统时间。

**示例：**
```bash
# 查看昨天/明天的日期
date -d "yesterday"
date -d "tomorrow"

# 查看过去或未来的时间
date -d "2 days ago"       # 2天前
date -d "next week"        # 下周
date -d "3 months 1 day"   # 3个月零1天后
date -d "last Friday"      # 上个周五

# 查看特定日期是星期几
date -d "1998-11-02" +"%A" # Monday
```

### 4. 查看文件的最后修改时间
使用 `-r` 选项可以查看特定文件的最后修改时间，而不是当前系统时间。
```bash
date -r /etc/passwd
```

### 5. 时间戳转换
在开发中经常需要在 Unix 时间戳和可读格式之间转换。
```bash
# 将当前时间转换为秒级时间戳
date +%s

# 将时间戳转换为可读格式 (注意：MacOS 下语法不同，这里是 Linux GNU date 语法)
date -d @1673587200
```

### 6. 设置系统时间 (需要 root 权限)
使用 `-s` (或 `--set`) 选项可以修改系统时钟。**注意：这通常会影响正在运行的服务，请谨慎操作。**
```bash
# 设置完整的日期和时间
sudo date -s "2026-01-13 11:30:00"

# 只设置日期
sudo date -s "2026-01-13"

# 只设置时间
sudo date -s "11:30:00"
```

### 7. 时区相关
*   **显示 UTC 时间：** 使用 `-u` 选项。
    ```bash
    date -u
    ```
*   **显示特定时区的时间：** 通过临时设置 `TZ` 环境变量。
    ```bash
    TZ="America/New_York" date
    TZ="Asia/Tokyo" date
    ```

### 8. 实用小技巧：在脚本中自动生成文件名
这是 `date` 命令在服务器运维中最常见的用途之一：
```bash
# 创建一个带日期后缀的备份文件
tar -czf backup_$(date +%F).tar.gz /path/to/data/
# 生成的文件名类似：backup_2026-01-13.tar.gz
```

### 总结常用参数速查：
| 选项 | 说明 | 例子 |
| :--- | :--- | :--- |
| `-d` | 显示字符串指定的日期 | `date -d "next Mon"` |
| `-u` | 显示 UTC (世界标准时间) | `date -u` |
| `-r` | 显示文件的最后修改时间 | `date -r filename` |
| `-s` | 设置系统时间 (root) | `sudo date -s "12:00"` |
| `+%s` | 输出 Unix 时间戳 | `date +%s` |
| `+%F` | 输出完整日期 | `date +%F` (2026-01-13) |
| `+%T` | 输出完整时间 | `date +%T` (10:55:00) |
